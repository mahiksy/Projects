How To Play

1. First choose one of the moves you want to play.

2. click on Fight button to start the game.

3. Next Round button appears when the result of the game displays. click on the button to start the next round.

4. who wins 3 rounds is the winner and when we have a winner, Play Again button appears which allows you to start the new game.
